# Webscraping-of-Meteorological-Data

Vous êtes data scientist, nouvellement recruté pour gérer tous les sujets relatifs aux données dans une entreprise qui prévoit la météo d'un certain nombre de villes. Mais l'entreprise étant nouvelle elle ne possède pas de données historisées sur la météo. Mais en revanche, elle est partenaire d'une structure qui présente sur sa plateforme les données historisées de la météo sur plusieurs périodes.

 Le partenariat suppose que pour la mise à disposition des données à votre entreprise, elle va devoir débourser une somme énorme. Les finances de l'entreprise ne permettent pas d'honorer cette condition. 
 
 Elle décide donc que vous fassiez usage de vos compétences d'expert en extraction de données.


Votre mission, si toutefois vous l'acceptez, est de collecter les données de la météo sur la plateforme de votre partenaire par les techniques de **web scraping**.
